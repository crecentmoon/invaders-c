## Setups(if you want to run on macOS)

- Install ncurses with brew

    ```
    brew install ncurses
    ```

- Compile and run binary

    ```
    cd "/Users/tsukiokayuu/Documents/GitHub/invaders-c" && gcc -o invaders main.c -lncurses && ./invaders
    ```
